Описание программы
Random Password Generator — это десктоп‑приложение с графическим интерфейсом для создания надёжных случайных паролей с настраиваемыми параметрами. Программа помогает быстро генерировать безопасные комбинации символов для защиты аккаунтов, сервисов и приложений, избавляя пользователя от необходимости придумывать пароли вручную.

Программа решает следующие задачи:

создание паролей заданной длины (от 4 до 32 символов);

настройка состава пароля (цифры, буквы, специальные символы);

мгновенное копирование сгенерированного пароля в буфер обмена;

ведение истории генераций с сохранением между запусками;

валидация ввода и обработка ошибок с понятными уведомлениями.

Требования для запуска
Что нужно установить на компьютер:

Python 3.6 или выше;

стандартные библиотеки Python (не требуют отдельной установки): tkinter, random, string, json, os, datetime.

Как запустить
bash
git clone https://github.com/твой-логин/random-password-generator.git
cd random-password-generator
