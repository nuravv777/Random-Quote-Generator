# Random-Quote-Generator
# Random Quote Generator (Генератор случайных цитат)

**Автор:** [Харисова Нурания]  
**Язык разработки:** Python 3.x  
**Библиотеки:** Tkinter (GUI), JSON, Random

## 📝 Описание проекта
Данное приложение представляет собой графический интерфейс (GUI) для генерации случайных вдохновляющих цитат. Проект создан в рамках учебного задания и демонстрирует работу с коллекциями данных, фильтрацией, обработкой событий и сохранением состояния в формат JSON.

## 🚀 Основные возможности
1. **Генерация:** Вывод случайной цитаты по нажатию кнопки.
2. **Фильтрация:** Возможность поиска цитат по конкретному **автору** или **теме**.
3. **История:** Ведение списка ранее просмотренных цитат в интерфейсе.
4. **Сохранение данных:** Автоматическое сохранение истории в файл `history.json` и её загрузка при повторном запуске.
5. **Валидация:** Обработка случаев, когда цитаты по заданным фильтрам не найдены.

## 🛠 Инструкция по запуску
1. Убедитесь, что у вас установлен Python версии 3.6 или выше.
2. Скачайте файл `import tkinter as tk.py` из этого репозитория.
3. Запустите приложение через терминал или командную строку:
   ```bash
   python import tkinter as tk.py
   ```

## 📂 Структура проекта
* `import tkinter as tk.py` — основной код приложения.
* `history.json` — файл с историей (создается автоматически после первого запуска).
* `README.md` — описание проекта.

## 🧪 Примеры использования (Тесты)
* **Тест 1 (Генерация):** Оставьте поля фильтров пустыми и нажмите кнопку — отобразится случайная цитата из общего списка.
* **Тест 2 (Фильтр):** Введите в поле "Тема" слово `IT`. Нажмите кнопку — программа выдаст цитату, связанную с программированием.
* **Тест 3 (Проверка истории):** После генерации цитаты закройте приложение и откройте снова. Список в нижней части экрана должен отобразить ваши предыдущие просмотры.
